Repository ini berisi kumpulan jawaban dari bootcamp Sanbercode untuk materi Automation Testing menggunakan Cypress, dan seluruh script dibuat oleh Azriel FachrulRezy.

Untuk materi 16, jawaban ada pada folder E2E -> InterceptLogin.cy.js di folder E2E yang berisi implementasi intercept pada proses login OrangeHRM

Untuk materi 17, tugas dikerjakan menggunakan format Page Object Model (POM). Struktur yang digunakan sebagai berikut:
1. Fixtures → loginData.js (Menyimpan seluruh data login yang digunakan dalam automation)
2. Support → pageObjeects → LoginPage.js (Menyimpan element-element yang digunakan pada proses pengujian, termasuk action yang dipanggil oleh script dan URL visit ke halaman login OrangeHRM)
3. E2E → POM → LoginPOM.cy.js (Menyimpan script automation utama yang menjalankan semua test case menggunakan format POM)

Untuk Quiz 3, jawabannya berada dalam folder E2E -> Login.cy.js yang berisi test case login
